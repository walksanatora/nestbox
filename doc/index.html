Functions

new (bios, The)
Creates a new box.

Parameters:

  • bios string or nil The BIOS to use. If unset, use loadBIOS later.
  • The table or nil disk data. If unset, defaults to an empty disk.

Returns:

    yellowbox A new box instance.
makeenv ()
Creates the environment for the box. This is mostly an internal function, but it's exported as part of the class.

Returns:

    table A new environment for the box.
getfenv ()
Returns the global environment for the box.

Returns:

    table The box's global environment.

Loads the BIOS if necessary and sets a new environment.

Resumes the box's execution, or starts it if it's not running.

Queues an event inside the box.

Parameters:

  • event string The event name to queue.
  • ... The event's arguments.
halt ()
Halts the box, deleting any coroutines and resetting the state.
mount (innerPath, outerPath)
Mounts a path from outside the box inside the environment.

Parameters:

  • innerPath string The path to the mount inside the box.
  • outerPath string The path to the mounted folder outside the box.
unmount (innerPath)
Unmounts a previously mounted directory.

Parameters:

Loads a BIOS file from a path.

Parameters:

  • path string The path to the BIOS.
enableCompression (enable)
sets the status of compression for disk files this uses LibDeflate [LibDeflate](https://github.com/MCJack123/CC-Archive/#libdeflate) this option only appears if LibDeflate was able to be required via `require("LibDeflate")`

Parameters:

  • enable boolean whether or not to compress vfs files
compressionEnabled ()
gets the status of compression for disk files this option only appears if LibDeflate was able to be required via `require("LibDeflate")`

Returns:

    boolean compression_enabled whether or not compression is enabled
loadVFS (path, readOnly)
Loads a VFS disk and optionally sets up write-backs. This uses a basic serialized table to store data. More efficient methods are available, but these are left to the user to implement.

Parameters:

  • path string The path to the VFS.
  • readOnly boolean or nil Whether the disk is read-only (won't sync changes to the file).
exportPeripheral (name, innerName)
Exports a peripheral from outside the box in.

Parameters:

  • name string The name of the peripheral as available outside the box.
  • innerName string or nil The name of the peripheral as it will appear inside the box. Defaults to the same name as outside.
